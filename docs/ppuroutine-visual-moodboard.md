# Ppuroutine Visual Moodboard

## Direction

뿌루틴의 비주얼은 **1-bit / low-resolution pixel mascot + clean care UI**로 잡는다.

앱 전체를 게임처럼 만들기보다는, 기능 화면은 깔끔하고 실용적으로 유지하고 뿌나 캐릭터와 작은 상태 아이콘에 도트 감성을 넣는다.

## Keywords

- tiny pixel
- cloud-like white dog
- calm senior pet
- warm care
- simple health log
- cozy but useful

## Character Style

뿌나 캐릭터의 핵심 특징:

- 흰색 구름 같은 털
- 옆으로 내려오는 둥근 귀
- 큰 검은 눈
- 작은 회갈색 코
- 작은 입
- 살짝 졸리고 얌전한 표정

도트 스타일 규칙:

- 16x16, 24x24, 32x32 느낌의 작은 스프라이트
- 검정 또는 짙은 초록 계열 픽셀 라인
- 털 디테일은 많이 그리지 않고 덩어리 실루엣으로 표현
- 너무 매끈한 안티앨리어싱 금지
- 도트 캐릭터는 앱 안의 장식, 상태, 빈 화면, 배지에 사용

## Color Palette

Primary:

- Pixel Ink: `#1F2D2B`
- Sage: `#8AB77A`
- Sage Dark: `#5E8C6A`
- Paper: `#F8F4EA`
- Ivory: `#FFFDF6`

Accent:

- Coral: `#F28F6B`
- Peach: `#FFE1D2`
- Yellow: `#F4C85A`
- Oat: `#D8C7A6`
- Warm Brown: `#8A6A4E`

Usage:

- `Sage` is the main brand color.
- `Pixel Ink` is used for pixel mascot and key text.
- `Coral` is only for important CTA or positive action.
- `Yellow/Oat` is used for cost, appointment, and gentle warning states.
- Backgrounds should stay warm and quiet.

## UI Changes From Current Prototype

1. Replace the current generic icon-like marks with pixel-inspired icons.
2. Use pixel Ppuna on onboarding, empty states, and small dashboard feedback.
3. Reduce bright multi-color emphasis; make Sage and Pixel Ink the dominant pair.
4. Keep cards clean with 8-12px radius.
5. Keep body typography plain and readable.
6. Do not make the whole UI pixelated. Pixel is an asset style, not the full interface style.
7. Keep the dark breathing timer, but add a small sleeping pixel Ppuna.
