# Ppuroutine Asset Prompts

## 1. Black And White Tiny Pixel Sheet

```text
Use case: stylized-concept
Asset type: pixel art mascot variation sheet for a pet-care app
Primary request: Create an original ultra-simple black-and-white pixel icon sheet of Ppuna, a small white Maltese-like senior dog. Use the provided Ppuna photos only for identity traits: fluffy white cloud body, floppy ears, big gentle dark eyes, small nose, calm sleepy expression. Use the first pixel example only as a broad mood reference for tiny monochrome pixel icons, not for copying exact characters or poses.
Style direction: very low resolution 1-bit pixel art, black pixels on white background, cute tiny virtual-pet icon feel, extremely simplified and readable, charmingly minimal, no grayscale, no color, no anti-aliasing.
Composition: one clean sheet with 12 separate tiny Ppuna icons arranged in a grid. Include: sitting front, sleeping curled, lying flat, eating from bowl, beside medicine, holding tiny notebook, hospital bag, happy bounce, tired face, looking up, tiny app icon head, tiny paw pose. Keep each icon original and distinct.
Technical constraints: each sprite should feel like 16x16 to 24x24 pixel art, crisp square pixels, no smooth lines, no text, no watermark, no background decoration, no direct copy of reference examples, no extra animals, no human characters.
```

## 2. Sage Retro Pixel Sprite Sheet

```text
Use case: stylized-concept
Asset type: pixel art mascot variation sheet for a pet-care app
Primary request: Create an original 1-bit retro pixel sprite sheet of Ppuna, a small white Maltese-like senior dog. Use Ppuna photos for identity: cloud-like white Maltese fur, floppy ears, gentle dark eyes, tiny nose, calm sleepy senior-dog personality. Use the second pixel example only as a broad mood reference for simple retro game sprites on a green background, not for copying the dog design.
Style direction: retro handheld game sprite, 1-bit dark ink pixels on muted sage green background, simple side-view and front-view dog sprites, crisp pixel grid, nostalgic but soft, readable at small size.
Composition: one sheet with 8 separated Ppuna sprites in consistent scale: idle side view, blink, tiny walk frame 1, tiny walk frame 2, sitting front, sleeping, holding care notebook, beside medicine bottle. Make the dog look fluffier and more cloud-like than the reference, with floppy ears and a gentle sleepy face.
Palette: muted sage green background, dark navy/forest pixel ink, off-white dog fill where needed. Very limited colors. No gradients. No smooth anti-aliasing.
Technical constraints: crisp square pixels, simple 32x32 or 48x48 sprite feeling, no text, no watermark, no direct copy of provided examples, no extra animals, no human characters, no 3D, no painterly shading.
```
